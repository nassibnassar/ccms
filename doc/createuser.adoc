==== create user

Define a new database user

[source,subs="verbatim,quotes"]
----
create user `*_user_name_*` with *_option_* '*_value_*' [, ... ]
----

[discrete]
===== Description

`create user` defines a new database user.

[discrete]
===== Parameters

[frame=none,grid=none,cols="1,2"]
|===
|`*_user_name_*`
|The name of the new user.

|`with ( *_option_* '*_value_*' [, ... ] )`
|Configuration options for the new user.
|===

[discrete]
===== Options

[frame=none,grid=none,cols="1,3"]
|===
|`encrypted password`
|Sets the cryptographic hash of the user's password.
|===

[discrete]
===== Examples

Create a user `anna`:

----
create user anna with encrypted password 'GSDDdftAsLcbPQXynZYBnM4ZT8wPMmdY9Ws3SZWVs2I';
----
