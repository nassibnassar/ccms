==== show

List the values of a system variable

[source,subs="verbatim,quotes"]
----
show filters

show funds

show projects [ archived ]

show project `*_project_name_*`

show sets [ in project `*_project_name_*` ]

show users
----

[discrete]
===== Description

`show` lists the current values of system configurations or other
variables.

[discrete]
===== Examples

----
show sets;
----
