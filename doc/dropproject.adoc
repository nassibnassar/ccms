==== drop project

Remove an archived project

[source,subs="verbatim,quotes"]
----
drop project `*_project_name_*`
----

[discrete]
===== Description

The `drop project` command removes an archived project from the
system.  This automatically drops the `object` table and deletes all
of its data.  All associated project properties are also removed.

[discrete]
===== Parameters

[frame=none,grid=none,cols="1,2"]
|===
|`*_project_name_*`
|The name of the project.
|===

[discrete]
===== Examples

Remove an archived project `myproj_20260610_103412`:

----
drop project myproj_20260610_103412;
----
