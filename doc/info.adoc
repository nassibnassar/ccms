==== info

Show supported commands

[source,subs="verbatim,quotes"]
----
info [ '*_topic_*' ]
----

[discrete]
===== Description

The `info` command shows information about supported commands.

[discrete]
===== Parameters

[frame=none,grid=none,cols="1,2"]
|===
|`'*_topic_*'`
|A command that the user would like information about.  If this
 parameter is not specified, a list of available topics will be shown.
|===

[discrete]
===== Examples

To show a list of available topics:

----
info;
----

To show information about the `select` command:

----
info 'select';
----
