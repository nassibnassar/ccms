alter project

Change the definition of a project

alter project project_name
    action

where action is one of:

    alter property property_name set value
    alter property property_name add subvalue
    alter property property_name drop { subvalue | all }

Description

The alter project command changes the definition of a project.

The alter property clause changes a property in the project’s metadata. The set
action is used for single-value properties, while add and drop are for
composite properties.

Parameters

project_name  The name of the project.

property_name The name of the property in the project metadata.

value         The value of a single-value property.

subvalue      A subvalue in a composite property.

Properties

  Property       Type                            Value
title        single-value text

action       single-value action name (acquire, retire, digitize, move, other)

mou_link     single-value text

funds        composite    fund name

origins      composite    location name (clockss, lehigh, nyu)

destinations composite    location name

tracks       composite    track name (offsite, reserve, stacks)

Examples

Set the title of project myproj to My Project:

alter project myproj alter property title set 'My Project';

Add a fund palci_cultural to the project:

alter project myproj alter property funds add palci_cultural;
