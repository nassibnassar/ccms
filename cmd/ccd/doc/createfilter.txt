create filter

Define a new filter

create filter filter_name
    where search_condition

Description

The create filter command defines criteria for selecting objects. These
criteria can then be used for querying sets via the filter() operator.

Parameters

where

The where clause specifies a search condition in the form of a Boolean
expression. Only objects that satisfy this condition will be selected.

See also Reference > Queries.

Examples

To filter on records where decision = true:

create filter decided where decision = true;

select * from p.s where filter(decided);
